# SNW Art Platform

Welcome to the SNW Art Platform! This project is a dedicated space for artists and enthusiasts to explore techniques, discover supplies, and connect with a creative community.

## Overview

This website serves as a comprehensive hub for art education and community engagement. It features a modern, dark-themed interface designed to be visually immersive.

### Key Features

*   **Art Techniques**: In-depth guides and resources for various mediums including:
    *   Acrylic Painting
    *   Oil Painting
    *   Charcoal Drawing
    *   Digital Art & Illustration
    *   Pastels & Watercolors
*   **Shop**: A curated selection of art supplies and tools.
*   **Events**: Information on upcoming workshops, exhibitions, and community gatherings.
*   **Community**: A platform for artists to share, discuss, and grow together.
*   **FAQ**: Quick answers to common questions about the platform and services.

## Getting Started

To explore the project locally:

1.  **Clone the repository** (if applicable) or download the source files.
2.  **Open `index.html`**: Simply navigate to the project folder and double-click `index.html` to open it in your preferred web browser. No server setup is required for the basic static content.

## Technologies Used

*   **HTML5**: Semantic structure for all pages.
*   **CSS3**: Custom styling with a focus on responsive design and a cohesive dark theme.
*   **JavaScript**: Interactive elements for navigation and user experience.
